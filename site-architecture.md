# frankdenneman.nl Site Architecture

This document describes the architecture of the frankdenneman.nl Hugo site so future changes can be made without rediscovering the structure.

It is intended for:
- future maintainers
- AI assistants
- debugging and development

# AI Assistant Context

This repository contains the source code for **frankdenneman.nl**, a Hugo-based technical blog about AI infrastructure.

Key facts for AI assistants:

- Static site generator: **Hugo**
- Theme: **Anatole**, loaded via **Hugo Modules**
- Hosting: **GitHub Pages**
- Configuration file: `hugo.toml`
- Primary layout overrides: `layouts/partials/`
- Structured data is generated by: `layouts/partials/schema.html`
- Metadata and page head logic are handled by: `layouts/partials/head.html`
- Custom CSS is located at: `assets/css/custom.css`

Content conventions:

- Articles live under `content/`
- Two major article series exist:
  - `Understanding AI Memory`
  - `Architecting AI Infrastructure`
- Series landing pages:
  - `content/understanding-ai-memory.md`
  - `content/ai-infrastructure.md`

Front matter fields used in articles:

- `series`
- `series_order`
- `categories`
- `track`
- `concepts`
- `description`
- `date`

Custom metadata usage:

- `concepts` describe technical topics and are used for linking and schema keywords
- `track` groups related infrastructure topics (e.g. AI Memory, AI Infrastructure)

Structured data:

- `WebSite` schema is generated for the homepage
- `BlogPosting` schema is generated for articles
- Schema logic is implemented in `layouts/partials/schema.html`

Important rule:

When modifying metadata, schema, or head logic, prefer editing **local partials in `layouts/partials/`** rather than theme files from the Hugo module.

# Theme System

Theme is loaded through Hugo Modules.

[module]
  [[module.imports]]
    path = "github.com/lxndrblz/anatole"

Theme files are stored in the Hugo module cache.

Typical location:

~/Library/Caches/hugo_cache/modules/

Local files in the repository override theme files automatically.
